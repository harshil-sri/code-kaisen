## What is this?

Code Kaisen is a **real-time 1v1 multiplayer coding arena** styled after Jujutsu Kaisen.

Two players enter a **Domain Expansion** — a private matchmaking room. They're given the same coding problem. When you submit a correct solution, the **Six Eyes AI** judges your code and deals damage to your opponent's HP bar in real time. Last one standing wins.

Standard competitive programming, but make it cursed.

---

## The Architecture


| Layer | Tech | Role |
|-------|------|------|
| ⚔️ Frontend | React + Vite | The arena UI — HP bars, code editor, live updates |
| 🌐 Backend | Node.js + Express + Socket.io | Domain server — rooms, WebSockets, HP state |
| 🧠 Six Eyes AI | Python + FastAPI + CodeBERT | Semantic code grading + damage calculation |

Questions are dynamically generated by an LLM (Groq/Gemini) — no static problem bank.

**What's working right now:**
- Players can connect and enter named rooms (`join_domain`)
- The server tracks both players' HP per room in real time
- Attacks fire over WebSockets and HP is deducted live
- Basic React client receiving and displaying server state
